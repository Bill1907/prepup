## 🎨 디자인 및 UI

- **스타일**: shadcn/ui (New York 스타일)
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 완벽 지원
- **다크 모드**: 지원
- **아이콘**: Lucide React
- **컬러 테마**: 블루-퍼플 그라데이션

## 🛠️ 기술 스택

- **프레임워크**: Next.js 15.4.6
- **UI**: React 19.1.0
- **스타일링**: Tailwind CSS 4
- **인증**: Clerk
- **배포**: Cloudflare
- **언어**: TypeScript

## 🚀 실행 방법

1. **개발 서버 시작**:
```bash
npm run dev
```

2. **브라우저에서 열기**:
```
http://localhost:3000
```

3. **빌드**:
```bash
npm run build
```

4. **배포**:
```bash
npm run deploy
```

## 🔐 환경 변수 설정

`.dev.vars` 파일에 Clerk 키 추가 필요:
```
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_key
CLERK_SECRET_KEY=your_secret
```

## 💡 팁

1. **테마 색상 변경**: `app/globals.css`에서 CSS 변수 수정
2. **컴포넌트 추가**: `npx shadcn@latest add [component-name]`
3. **라우트 보호**: `middleware.ts`에서 보호된 라우트 추가
4. **메타데이터 수정**: 각 페이지의 `metadata` 객체 수정
